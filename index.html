Info

In diesem Projekt wurde die OCR.space API angebunden. Die Herausforderung war es, die Geräte-Kamera einzubinden, einen definierten Bereich im Kamera-Stream als Rechteck zu definieren und diesen auch explizit zu erfassen.

Es wird ein 9-stelliger Code erwartet. Erlaubt sind Buchstaben von A-Z und Zahlen von 1-9. Der Code kann in beiden Formaten "XXX XXX XXX" oder "XXXXXXXXX" eingegeben werden.
